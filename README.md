MdzEvents

==========
**Setup**

1- npm install

2- sudo npm install -g grunt-cli

3- sudo npm install -g bower

4- bower install

**Run**

grunt serve (browser)

grunt platform:add:android

grunt run:android

grunt run:android -l -c -s (server live)
